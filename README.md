# VINEOSAUR

## Requirements
1. NodeJS
2. brunch - `npm install -g brunch`, run as `sudo` in linux
3. bower - `npm install -g bower`, run as `sudo` in linux

### Setup

    git clone https://github.com/mikechau/vineosaur.git
    cd vineosaur
    npm install && bower install

### Getting Started

    brunch watch --server
